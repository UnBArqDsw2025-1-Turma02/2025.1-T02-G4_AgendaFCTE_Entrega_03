## Como rodar

O código cria 1 objeto Facade e faz a inscrição de um usuário em um evento (processo que envolve várias classes). Para rodar, basta dar os comandos abaixo na raiz do projeto

```
cd docs/assets/Facade/codigo
javac Main.java
java Main
```
